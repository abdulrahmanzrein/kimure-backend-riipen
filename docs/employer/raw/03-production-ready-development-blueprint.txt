1.4 — Environment & Security

All API keys (Gemini, Equifax, Stripe) stored as ENV variables — never in front-end code
HTTPS enforced on all endpoints — no HTTP fallback
Role-Based Access Control (RBAC) enforced at API middleware level
GDPR + PIPEDA + CASL compliance built into user data models from Sprint 1
All passwords hashed with bcrypt (minimum 12 salt rounds)
Rate limiting on all public API endpoints (prevent abuse)
Automated daily database backups to AWS S3

4.2 — Backend AI Orchestrator Architecture

A central AI Gateway Service handles all AI requests. This service receives user queries, enriches them with profile and market data, calls Gemini API, caches the response, logs the interaction, and returns a structured result.

AI Gateway Service
  ├── Property Scout Engine  →  GET /ai/scout
  ├── Investment Analyzer Engine  →  POST /ai/analyze
  ├── Rental Matching Engine  →  POST /ai/rental
  ├── Valuation Engine  →  POST /ai/valuate
  └── Mortgage Engine  →  POST /ai/mortgage


4.3 — Gemini API Integration Process

Step 1: Developer accesses Google AI Studio (aistudio.google.com) with Kimure company Google account
Step 2: Generate API Key — stored ONLY in backend ENV variables, never in front-end code
Step 3: Extract exact prompt instructions from each of the 5 Gemini Gems
Step 4: Code prompts into the backend AI service, injecting dynamic user data at runtime
Step 5: Each AI call to Gemini injects: user budget, profile data, property data, market comparables
Step 6: Gemini response is parsed, structured into a report card, and returned to UI
Step 7: All interactions logged in MongoDB for AI memory and personalization

7.2 — Primary Revenue Model: Cost-Per-Lead Marketplace

How It Works
Agent loads a monthly budget into Kimure
Agent selects target postal codes / ZIP codes
When a qualified user in that area clicks Lead Button, agent is charged
Higher budget = higher ranking on "Recommended Agents" list
Industry average CPL for real estate: $50 to $400 per lead
Kimure leads are pre-qualified by AI — warranting premium pricing
Agent dashboard tracks spend, leads received, conversion rate
Pay-at-Closing Option (Flex Model)
Lead provided at zero upfront cost
Agent signs agreement to pay 25–35% referral fee on gross commission at closing
Ideal for new agents building their pipeline
Kimure tracks via transaction reporting in the CRM
Commission auto-calculated by platform on deal closure

These two models can be offered in parallel. Agents choose their preferred structure.
